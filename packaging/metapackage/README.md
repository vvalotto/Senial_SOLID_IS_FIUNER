# Senial SOLID FIUNER - Sistema de Procesamiento de Señales

Sistema completo de procesamiento de señales digitales con los **5 principios SOLID aplicados** (SRP, OCP, LSP, ISP, DIP). Caso de estudio del libro *"SOLID: La Arquitectura que Emerge del Dolor"*.

## Instalación

```bash
pip install senial-solid-fiuner
```

## Uso

```bash
# Comando directo
senial-solid-fiuner

# O como módulo
python -m lanzador.lanzador
```
